# Audio Assets

Drop engine audio files here. The app falls back to Web Audio API synthesis if files are missing.

| File | Description |
|------|-------------|
| `bmw-e39-m5.mp3` | BMW S62 V8 rev — deep raspy 5.0L growl |
| `porsche-997.mp3` | Porsche M97 Flat-6 wail — metallic high-revving bark |
| `toyota-sienna.mp3` | Toyota 2GR-FE V6 — smooth refined hum |
| `kia-sportage-phev.mp3` | Kia G4FJ PHEV — electric whir transitioning to 4-cyl idle |

Recommended: 128kbps mono MP3, 2–3s duration, normalized to -12 dB LUFS.
